# Welcome to ZeroStarter

ZeroStarter is a production-ready SaaS starter template that combines modern web technologies with battle-tested architectural patterns. Built with a monorepo structure, it provides end-to-end type safety, modular packages, and a complete development workflow out of the box.

<Note>
  ZeroStarter is currently in **Release Candidate (RC)** status. All implemented features are stable and production-ready. We're actively adding new features and integrations.
</Note>

## Key Features

### Type-Safe API Client

ZeroStarter uses [Hono RPC](https://hono.dev/docs/guides/rpc) to provide end-to-end type safety between backend and frontend:

<CodeGroup>
```typescript Backend (api/hono/src/index.ts)
import { Hono } from "hono"

const app = new Hono()
  .basePath("/api")
  .get("/health", (c) => {
    const data = { message: "ok", version: "1.0.0" }
    return c.json({ data })
  })

export type AppType = typeof app
```

```typescript Frontend (web/next/src/lib/api/client.ts)
import type { AppType } from "@api/hono"
import { hc } from "hono/client"

const honoClient = hc<AppType>("http://localhost:4000")
export const apiClient = honoClient.api

// Fully typed request and response
const res = await apiClient.health.$get()
const { data } = await res.json() // TypeScript knows the shape!
```
</CodeGroup>

### Clean Monorepo Structure

```
.
├── api/
│   └── hono/          # Backend API server (Hono)
├── web/
│   └── next/          # Frontend application (Next.js)
└── packages/
    ├── auth/          # Shared authentication logic (Better Auth)
    ├── db/            # Database schema and Drizzle configuration
    ├── env/           # Type-safe environment variables
    └── tsconfig/      # Shared TypeScript configuration
```

### Modern Tech Stack

- **Runtime**: [Bun](https://bun.sh) for blazing-fast performance
- **Build System**: [Turborepo](https://turbo.build) for optimized monorepo builds
- **Frontend**: [Next.js 16](https://nextjs.org) with React 19
- **Backend**: [Hono](https://hono.dev) for lightweight, fast APIs
- **Database**: [PostgreSQL](https://www.postgresql.org) with [Drizzle ORM](https://orm.drizzle.team)
- **Authentication**: [Better Auth](https://better-auth.com) with social providers
- **UI Components**: [Shadcn UI](https://ui.shadcn.com) with [Tailwind CSS](https://tailwindcss.com)
- **Data Fetching**: [TanStack Query](https://tanstack.com/query/latest)
- **Validation**: [Zod](https://zod.dev) for runtime type safety
